# cursor_template

Quick start for cursor dev, including: 
- .gitignore 
- venv setup
- requirements.txt 

### Setup Environment
1. Create a virtual environment named `myenv`:
```bash
python -m venv myenv
```

2. Activate the virtual environment:
    - On Unix or MacOS:
        ```bash
        source myenv/bin/activate
        ```

3. Install required packages:
```bash
pip install -r requirements.txt
```
